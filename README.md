# GRC Risk Assessment
Mock cybersecurity risk assessment report identifying key risks, evaluating impact and likelihood, and recommendations of mitigation strategies using GRC principles.

This project analyzes potential cybersecurity threats by evaluating their likelihood and impact, then proposes mitigation strategies aligned with industry best practices. 

Key Areas
- Weak authentication practices
- Exposure to sensitive customer data
- Phishing and social engineering
- Third-party platform risks
- Lack of data backups

Skills Demonstrated
- risk identification and analysis
- Risk evaluation
- Mitigation strategies development
- Understanding GRC principles
- Documentations and reporting
